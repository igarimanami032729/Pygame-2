# 🔥 귀멸의 칼날: 렌고쿠의 칼날 (Rengoku's Yaiba)

> Python + Pygame 기반의 귀멸의 칼날 테마 게임 
> 렌고쿠 쿄쥬로를 조종하여 하늘에서 떨어지는 음식들을 먹고,  
> 오니(아카자)의 공격을 피하거나 베어내며 최대한 오래 생존하는 아케이드 액션 게임입니다.

---

## 게임 스토리

무한열차 전투 이후, 렌고쿠 쿄쥬로는 사후의 세계에서  
다시 한 번 아카자와 맞서 싸우게 됩니다.  
하늘에서 떨어지는 영혼의 음식으로 체력을 회복하며  
끊임없이 떨어지는 오니들을 베어내야만 합니다.  

렌고쿠의 생명은 세 개의 불꽃(❤️)으로 표현됩니다.  
이 불꽃이 모두 꺼지면, 그 불꽃은 사라지고 게임이 끝납니다.  
그러나 음식의 영혼을 흡수하면 다시 불꽃이 피어납니다.

## 🎮 조작법

| 키 | 기능 |
|----|------|
| ← / → | 렌고쿠 좌우 이동 |
| Space | 공격 (짧은 시간 동안 칼 베기 모션) |
| R | 게임 재시작 |
| ESC | 게임 종료 |

> 💡 공격 중일 때 오니와 부딪히면 베어내기 성공 (점수 +50점)  
> 💥 공격하지 않을 때 오니와 부딪히면 하트 -1개

## 🍱 게임 규칙

### 1️⃣ 음식 (Food)
- 3종류의 음식(`food1.png`, `food2.png`, `food3.png`)이 랜덤하게 하늘에서 떨어집니다.  
- 렌고쿠가 음식을 먹으면 점수가 올라갑니다.  
- 특정 음식(`food2.png`)은 체력(❤️)을 1개 회복시킵니다.  
- 일정 확률로 고득점 아이템(`food3.png`)이 등장합니다.

| 이미지 | 효과 | 점수 |
|--------|------|------|
| 🍙 `food1.png` | 일반 음식 | +10 |
| 🍢 `food2.png` | 회복 음식 | ❤️ +1 |
| 🍡 `food3.png` | 고급 음식 | +20 |

---

### 2️⃣ 오니 (Akaza)
- 오니(`akaza.png`)는 랜덤 위치에서 떨어집니다.  
- 렌고쿠가 공격하지 않은 상태에서 닿으면 ❤️ -1개  
- ❤️가 0개가 되면 게임 오버 화면이 표시됩니다.  
- 스페이스(공격) 중 오니와 닿으면 “베어내기 성공”으로 +50점 획득  
- 레벨이 오를수록 오니의 낙하 속도와 등장 빈도가 증가합니다.

---

### 3️⃣ 하트 (Heart)
- 렌고쿠는 최대 ❤️ 3개를 가집니다.  
- 화면 좌측 상단에 하트 이미지(`./image/heart.png`)로 표시됩니다.  
- 음식으로 회복하거나 오니에게 맞으면 감소합니다.

---

### 4️⃣ 점수 및 레벨 시스템
- 점수가 200점 이상이 될 때마다 자동으로 레벨이 1 증가합니다.  
- 레벨이 오를수록 오니의 낙하 속도가 빨라집니다.  
- 음식 등장 속도도 약간 빨라집니다.  
- 레벨은 게임 화면 오른쪽 상단에 표시됩니다.
